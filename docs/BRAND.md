# Brand Notes

AiSH is a Dawnlight Labs pilot project.

## Naming

Use `AiSH` in product copy and headings.

Avoid inconsistent variants such as:

- AISH
- Aish
- AI Shell, unless explaining the concept rather than the product name

## Company attribution

Use `Dawnlight Labs` for creator, publisher, metadata, and installer attribution.

Suggested footer attribution:

```text
© 2026 Dawnlight Labs
```

## Product positioning

AiSH should be described as:

- an AI-native provider shell
- a terminal AI layer
- a local-first shell assistant where possible
- a command-planning and approval-gated execution tool

Avoid describing current `main` as a Tauri desktop app. The old desktop app is archived on the `app-provider-archive` branch.

## Visual direction

The current README uses the existing horizontal SVG brand assets from `apps/site/public/brand/`. Preserve those assets unless a deliberate brand refresh is being made.
