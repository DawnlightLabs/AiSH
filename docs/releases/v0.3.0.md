# AiSH 0.3.0 — First stable release

AiSH 0.3.0 is the first stable provider-shell release from Dawnlight Labs.

- AiSH now checks for a newer stable release at most once every 24 hours when the shell starts and asks before installing it.
- Updates replace the current platform binary directly; uninstalling and reinstalling AiSH is not required.
- Windows updates refresh Start menu, App Paths, and Installed apps registration after the new binary is applied.
- Added `aish --uninstall` and `aish --uninstall --yes` for complete user-level removal.
- Windows uninstall removes AiSH from PATH, Start menu, Installed apps, Windows Terminal, and supported editor terminal profiles.
- Fresh Windows installs continue to work without administrator privileges.
- Manual `/update` and `aish --update` checks remain available.

## Upgrade from 0.2.x

Existing 0.2.x installations do not contain the new automatic prompt yet. Run `/update` inside AiSH or `aish --update` once to install 0.3.0. After restarting AiSH, future releases will be detected automatically.

## Install

Windows PowerShell:

```powershell
irm https://aish.dawnlightlabs.com/install.ps1 | iex
```

macOS and Linux:

```bash
curl -fsSL https://aish.dawnlightlabs.com/install | bash
```

## Uninstall

```bash
aish --uninstall
```

Models stored outside the AiSH installation directory are preserved during uninstall.
