# AiSH v0.4.1

Windows installer reliability patch.

## Fixed

- Runs setup from the extracted staging executable instead of asking the installed executable to overwrite itself while it is running.
- Repairs empty Windows Terminal settings files, including UTF-8 BOM-only files.
- Backs up malformed Windows Terminal settings before rebuilding the minimum valid structure required for the AiSH profile.
- Preserves the branded AiSH icon in Windows Terminal after registration repair.

## Upgrade

Users on AiSH 0.3.1 or 0.4.0 can run:

```text
aish --update
```

For a failed or incomplete 0.4.0 installation, rerun the installer after closing existing AiSH sessions:

```powershell
irm https://github.com/DawnlightLabs/AiSH/releases/latest/download/install.ps1 | iex
```
