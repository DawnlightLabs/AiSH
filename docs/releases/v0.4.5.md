# AiSH v0.4.5

This release fixes clean-install local AI execution on Windows, macOS, and Linux.

## Fixed

- Removed developer-machine defaults that pointed to `Downloads/aish-model` and `Downloads/llama.cpp`.
- Installed the GGUF model under AiSH's managed `models` directory.
- Bundled the platform `llama-cli` runtime and its required libraries inside release archives.
- Installed the runtime under AiSH's managed `runtime` directory.
- Migrated known legacy Downloads-based paths automatically at startup.
- Added runtime bootstrap support for existing installations upgraded through `/update`.
- Updated the Windows installer to verify and persist both managed model and runtime paths.

## Validation

The release passed CI, Planner Contract, AI Run Routing Contract, Windows Updater Smoke, PowerShell syntax checks, Windows lifecycle tests, and workspace checks on Ubuntu and Windows.
