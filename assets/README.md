# AiSH logo pack

## Core variants

- `full-horizontal`: icon + AiSH + “Artificially Intelligent Shell”
- `wordmark`: AiSH only
- `wordmark-tagline`: AiSH + tagline
- `icon`: approved `^ _ ^` shell emblem
- `stacked`: vertical composition
- `app-icon-dark`: white emblem on near-black square
- `app-icon-light`: black emblem on warm-white square

## Recommended use

- Favicon: 16, 32, or 48 px icon
- Apple touch icon: 180 px app icon
- Android/PWA: 192 and 512 px app icons
- GitHub avatar: 512 or 1024 px icon
- Website navbar: 600w or 900w horizontal logo
- Retina website/print: 1600w or 2400w horizontal logo
- Open Graph: 1200×630
- GitHub social preview: 1280×640
- X/LinkedIn banner: 1500×500

Use black assets on light backgrounds and white assets on dark backgrounds.
